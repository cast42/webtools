Vitodens 300 – Granny-proof stappenplan

Doel: Codering 1 openen, d3 naar 12, en B0 + B2 noteren en tijdelijk op 0 zetten.

Knoppen – zo herken je ze

Knop Naam Wat doet die?
+ PLUS Volgende code / hogere waarde
− MIN Vorige code / lagere waarde
OK OK Open code / bevestig waarde
RADIATOR (rechts) Samen met kraan: codering openen/sluiten
KRAAN (links) Samen met radiator: codering openen/sluiten

Als je de icoontjes niet ziet: controleer of de PNG-bestanden in "docs/assets/" staan.

Altijd hetzelfde patroon

1️⃣ Zoeken
+ / −
2️⃣ Openen
OK
3️⃣ Veranderen
+ / −
4️⃣ Bewaren
OK (“Overgenomen”)
Regel 1: eerst noteren, dan pas op 0 zetten.

De echte stappen

A) Codering 1 openen

  1. Druk tegelijk op (links) + (rechts).
  2. Houd 2 seconden ingedrukt.
  3. Een code (bv. A0) knippert.

B) d3 op 12 zetten

  1. Druk op + of − tot d3 knippert.
  2. Druk OK.
  3. Zet de waarde op 12 met −.
  4. Druk OK ("Overgenomen").

C) Naar B0 en B2 (noteren → 0 → OK)

  1. Druk vaak op + tot B0 knippert (dit kan even duren).
  2. Druk OK zodat de waarde knippert.
  3. Schrijf de waarde op.
  4. Zet op 0 met + / −.
  5. Druk OK (Overgenomen).
  6. Druk weer vaak op + tot B2 knippert.
  7. Herhaal: OK → noteren → 0 → OK.

D) Codering afsluiten

  1. Druk opnieuw tegelijk op KRAAN + RADIATOR.
  2. Houd 1 seconde ingedrukt.
Na de test: zet B0 en B2 terug naar de originele waarden die je genoteerd hebt.
